# General Prompt Template with Input Variables

This is a general template for creating prompts that accept input variables. Copy this template when creating new prompts that need to accept variable inputs.

## Input Variables

- **VARIABLE_NAME**: Description of what this variable represents
- **ANOTHER_VARIABLE**: Another example variable
- **THIRD_VARIABLE**: A third example variable (optional)

## Instructions

Provide your instructions here, referencing variables as {{VARIABLE_NAME}}.

For example:

- Process the {{INPUT_TYPE}} according to the {{SPECIFICATION}}
- Create a {{COMPONENT_TYPE}} that implements {{FUNCTIONALITY}}
- Follow the {{STANDARD}} for implementation
- Output should be placed in {{OUTPUT_LOCATION}}

## Expected Output

Describe what the output should look like after processing the variables, including:

- Files to be created/modified
- Specific implementation details
- Testing requirements
- Documentation updates
